nvim-in-browser — Configuration

Edit your init.lua — it runs in a WASM sandbox.

This runs in a browser sandbox. Pure-Lua/Vimscript config, keymaps, options, and pure-Lua plugins work. Config that spawns processes or uses the network (LSP servers, git, telescope+ripgrep, plugin managers that clone repos) will NOT work. Plugin loading (bundling pure-Lua plugins) is coming in a follow-up.
